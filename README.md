## 🚀 项目简介（About the Project）

> 简单看看😉

- **项目类型**：嵌入式、数字信号处理、PCB设计

- **主要功能**：检测电机振动三轴加速度、数据预处理、串口发送数据

- **适用场景**： 低频电机振动信号（采集传感器带宽1600HZ）
